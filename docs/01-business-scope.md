# Business Scope

TeamPulse represents a lightweight engineering team management platform. The business domain is team health, individual contribution, goals, feedback, and evaluation visibility.

## Roles

| Role | Purpose | Current Behavior |
| --- | --- | --- |
| Manager | Oversees teams and members | Can manage teams, members, evaluations, goals, feedback, and notes |
| Team Member | Views personal and team context | Can view own evaluations/goals/feedback and limited team/member views |

## Included Features

- Organization dashboard for managers.
- Personal dashboard for team members.
- Teams list and team details.
- Members directory and member profile.
- Evaluation management.
- Feedback tracking.
- Goals tracking.
- One-to-one notes on member profiles for managers.
- Role-aware business guide: How TeamPulse Works.
- Theme switching.
- Floating page help.
- Learning Lab with Angular learning, MCP Servers placeholder, and Run Locally placeholder.

## How TeamPulse Works Guide

The `How TeamPulse Works` page is a business/user guide available to both managers and team members.

For managers, it explains how to use team health, goals, evaluations, feedback, and risk signals as a practical review rhythm:

1. Start from Dashboard.
2. Review team health.
3. Check members and risks.
4. Update goals.
5. Review feedback and evaluations.

For team members, it explains how to use personal goals, feedback, evaluations, skill/profile information, and team health context:

1. Review dashboard.
2. Check own goals.
3. Read feedback.
4. Track evaluation trends.

The guide includes the sample-data notice used by public-facing TeamPulse screens.

## Excluded Features

- Real authentication or authorization server.
- Password hashing.
- JWT tokens.
- ASP.NET Identity.
- Database persistence.
- Entity Framework.
- File uploads.
- Notifications.
- Runtime AI generation or AI summaries.
- Multi-language UI.

The fake role model is only a workshop teaching device. It demonstrates role-based UI and route guards without adding production security concerns.
